# Soil Quality Index Analysis

This repository contains a reproducible analytical workflow for evaluating soil quality indicators in irrigated mango orchards in the Brazilian semiarid region.

The project is under development and currently uses private data stored outside version control.

## Project structure

- `data/raw/private/`: private raw datasets not tracked by Git
- `data/processed/private/`: private processed datasets not tracked by Git
- `scripts/`: Python scripts for data processing and analysis
- `notebooks/`: exploratory notebooks
- `figures/`: generated figures
- `tables/`: generated tables
- `docs/`: methodological notes and documentation

## Current analytical goals

- Reorganize the soil quality dataset for reproducible analysis
- Test clay-normalized indicators, including P Mehlich-1 and GMea
- Generate new figures for manuscript development
- Compare original and derived soil quality indicators
